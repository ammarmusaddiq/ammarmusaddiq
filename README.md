### 🗂️ Featured Automations

---

**🛒 E-Commerce Store Assistant**
> *Your store never sleeps — AI handles search, stock, orders & restock alerts, all on autopilot.*

An AI agent wired into Google Sheets that lets customers search products, book orders, and update stock via chat — and auto-emails alerts when inventory runs low.

---

**📧 AI Email Sorter**
> *Inbox chaos → structured data. Gmail triggers AI to classify, parse & log every email in seconds.*

Gmail fires on new emails → an LLM chain classifies and extracts key fields → structured output gets appended to a Google Sheet. Junk gets ignored. Signal gets captured.

---

**💬 Facebook Messenger AI Agent**
> *Text, image, or voice — one smart agent handles them all on Messenger, powered by Groq AI.*

A multi-modal Messenger bot that routes by message type: image analysis via Vision AI, text queries via Groq chat agent (with product search & order creation), and voice transcription → AI response. All replies flow back to the user in real time.

---

**📸 Instagram Reels Data Scraper**
> *From hashtag to dataset — scraping Reels data at scale with zero manual effort.*

Loops over target accounts, runs Apify actors to extract Reel metadata, waits for datasets to populate, then stores everything neatly into a structured record database — fully automated.
